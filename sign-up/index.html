Log in or create an account

There are two publicly available instances of KoBoToolbox, one for the general public and a separate one for the humanitarian community, hosted directly by the United Nations Office for the Coordination of Humanitarian Affairs. For more information on the differences between the two servers, please read our Which Server Should I Use? support article.

Note that it is also possible to install KoBoToolbox on your own server (or on a local machine) using Docker. See our kobo-install repository on GitHub for details.
